# ryopc Music Platform

ryopc Music Platform は、完全無料で構築された音楽再生・配信プラットフォームです。

ユーザーは、ローカルファイルやURLから音楽を再生できるほか、自作のフリーBGMをアップロードし、共有することができます。

本プロジェクトは、Cloudflare（Workers・R2・D1）およびGitHubを活用し、サーバーレスかつ高性能なインフラを実現しています。

## 主な機能
- PWA対応（インストール・オフライン再生）
- URL / ファイル再生対応
- 立体音響（左右パン・音響演出）
- イコライザー機能
- プレイリスト作成・共有
- フリーBGMアップロード・配信
- 通報・管理システム

## 特徴
- 完全無料で利用可能
- サーバーレス構成による高い可用性
- オープンソースによる継続的な改善
- フリーBGM限定で安全に利用可能

## Philosophy
ryopcは「完全無料・プロ級インフラ」を理念とし、
誰でも高品質な技術を利用できる環境を提供することを目指しています。

本プロジェクトはその一環として開発されています。
